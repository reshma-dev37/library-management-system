# library-management-system
A simple console-based Library Management System developed in C++.
This project demonstrates Object-Oriented Programming concepts and basic data handling.

Features:
Add new books
Delete books by ID
Search for books
Display all available books
Add library members
Issue a book to a member
Return a book

Concepts Used:
This project covers important C++ concepts:
Classes & Objects
Constructors
Encapsulation
Vectors for dynamic storage
Basic file structure in C++ (header + src files).
